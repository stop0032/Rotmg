Stop0032's Guide To Easiest Class To Max

Easiest class to max means the least amount of pots needed to max a perfectly average roll. To find out, we will find the average stat for each class and the max stat for each class, find the difference between each stat, and convert that difference into amount of def pots. Then we can compare which class takes the least def pots to max. We will use the current conversion rate to convert each stat difference into def pots
